# Control AC Unicaribe
Durante una prohibición temporal de tener los controles de los Aires acondicionados en los laboratorios del Edificio de Ingenierías y para evitar atravesar el edificio dos veces (una para obtener el control remoto y otra para devolverlo) opté por esta solución.
### Materiales
* [Sparkfun Max Power IR LED Kit](https://www.sparkfun.com/products/retired/10732) (_Descontinuado_) o Sensor IR compatible
* Arduino UNO
### Conexiones
Pin 12 a Emisor IR

### Uso
Por medio del monitor serie ingresar los comandos
* **on** Enciende el AC
* **off** Apaga el AC
* **sw** Enciende o apaga el swing del AC
